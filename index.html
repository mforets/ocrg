How to contribute

If you would like to contribute with an exercise please send me an email and I will upload it here. Otherwise you can directly do it yourself, you just need a github account (it is free).

To actually create new content, you can do it in any text editor and save the file in .md (Markdown) format. There exist as well several programs specially conceived to formatting in Markdown. On a personal note, my favorite (for MacOS) is the open-source project MacDown.

If you enjoy working online, check stack edit.
